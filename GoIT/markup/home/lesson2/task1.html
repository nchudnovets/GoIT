Семантические тэги.

Просто текст.

Тег - элемент HTML, представляющий из себя текст, заключенный в угловые скобки. Тег является активным элементом, изменяющим представление следующей за ним информации.
Тег может иметь некоторое количество атрибутов. Обычно используются два тега - открывающий и закрывающий.

Обозначить ошибку в тексте можно тегом del (по умолчанию он будет перечеркнут),
вот так:
перечеркнутый текст
а вставленный текст — тегом ins (обозначается подчеркиванием), вот так:
вставленный текст

Несмотря на то, что это строчные теги (не разрывают строки), внутри них допускается размещение блочных элементов.
Например, перечеркнутый абзац:
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Magnam incidunt officia vitae mollitia amet dolores ut modi,
autem obcaecati deleniti, minima quos asperiores qui quidem harum! Consequuntur molestias, modi sunt!
Или вставленный абзац:
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Consectetur unde cum, vel eos quam fugit ipsum voluptates architecto.
Voluptatum deleniti voluptate unde, ratione aspernatur consectetur ut repudiandae, quasi quo nulla.

Немного кода.

Если на страницу нужно вывести исходник HTML кода, то следует оформить это вот так:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Тег VAR</title>
</head>
<body>

<p>В данном примере делается запрос к базе данных для получения поля <var>content_title</var>
из таблицы <var>content</var>, причем должно удовлетворяться условие
<code>section_id = 1</code>.</p>

</body>
</html>

Если не сконвертировать угловые скобки в ascii знаки, то результатом вывода будет лишь текст абзаца:

В данном примере делается запрос к базе данных для получения поля content_title из таблицы content, причем должно удовлетворяться условие section_id = 1.
